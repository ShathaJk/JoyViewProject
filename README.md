# JoyView MVP - Safe Content System

## Project Description
This project is a Minimal Viable Product (MVP) for the JoyView system, demonstrating core functionality for parental control over video content. The MVP covers:
1. Parent Login (R1)
2. Channel Management (Add/Remove approved channels) (R2 & R3)

## Prerequisites
* Java Development Kit (JDK) - Version 20 or newer
* IntelliJ IDEA

## Setup and Running Instructions (Crucial Step)

### 1. Project Path Correction 
The project folder must be placed in a simple directory (e.g., C:\JoyViewProject) to avoid runtime errors.

### 2. Running the Application
1. Open the project folder in IntelliJ IDEA.
2. Navigate to the main file: `src/app/JoyViewApp.java`.
3. Run the `main` method by clicking the green run button.
